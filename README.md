## 執筆の方針

定義や命題にはできるだけその概要や導入の動機、応用先について一言程度添える。
演習問題にはできるだけ詳細な解答をつける。

命題・補題・定理・系の環境の使い分けは次の通りとする。
基本的には命題環境を用いる。
有名な定理には定理環境を使う。
証明を分割するほど長い命題にも定理環境を使い、
その際に切り出す命題には補題環境を使う。
定義の well-defined 性のために必要な命題にも補題を使う。
系は定理の直後に用いる。
